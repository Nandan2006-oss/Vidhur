# Vidhur
Vidhur is a personal AI agent built from scratch using Python. The goal of this project is to understand how AI agents actually work beyond just using APIs.

It starts as a basic chatbot and is gradually extended to support features like context awareness, command execution, memory, and tool integration. Over time, the agent will be able to perform tasks, store information, and assist the user more intelligently.

This project focuses on learning core concepts such as system design, handling user input, integrating AI models, and building scalable agent architectures.
